# Typing Level Tracker

文字を打つたびに経験値が貯まり、レベルアップするVSCode拡張機能です。

## 機能

- タイピングするたびにXPを獲得
- ステータスバーにレベルと進捗を表示
- クリックで詳細XPを表示

## 経験値テーブル
100 + Math.floor(level / 5) ** 3
